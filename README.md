# DataMining4-GraphSpectra
Implement and test the spectral graph clustering algorithm analysing two kind of sample graphs  

## Run
`main.m`  

## Dataset
`example1.dat`: A real graph -  This data set was prepared by Ron Burt. He dug out the 1966 data collected by Coleman, Katz, and Menzel on medical innovation. They collected data from physicians in four towns in Illinois: Peoria, Bloomington, Quincy, and Galesburg.  
`example2.dat`: A synthetic graph
